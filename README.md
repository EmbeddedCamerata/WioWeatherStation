## Weather Station for Wio Terminal

### Dependencies

1. 在 `.ino`中**配置您的WIFI**(SSID + PASSWORD)，所需要获取的城市**Location ID**（默认北京）；
2. 需要通过TF卡导入CourierNewPS-ItalicMT-16， TimesNewRomanPSMT-24字体及主页图片，天气图标等。该项资源在 `./source `内；
3. Enjoy 😆
